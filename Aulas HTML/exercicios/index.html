TAG h1

As tags <h> são usadas para inserir títulos, indo de h1, a maior e mais importante semanticamente, até h6, a menor. Como mostrado abaixo:

h2

TAG p

A tag <p> é usada para inserir parágrafos.

TAG br

A tag <br>é usada para quebra de linha na escrita de textos.

TAG input

A tag <input> é usada para que o usuário insira a informação solicitada. Exemplo:
Insira seu nome:

TAG i

A tag <i> é usada para colocar o texto em itálico.

Tag b

A tag <b> é usada para colocar o texto em negrito.

TAG strong

A tag <strong> tem um significado sêmantico importante para leitores de tela com deficiencia visual, como se evidenciando a informação.

TAG li

A tag <li> é usado para criar itens de listas.

TAG ul

A tag <ul> é usada para listas desordenadas.Exemplo:
  • Item A
  • Item B
  • Item C
  • Item D

TAG ol

A tag <ol> é usada para listas ordenadas.Exemplo:
  1. Item E
  2. Item F
  3. Item G
  4. Item H

TAG dl

A tag <dl> é usada para criar listas de definição, para isso usa-se <dt> para marcar o item e <dd> para a explicação.

TAG a

A tag <a> é usada para inserir hiperlinks, ancorando até a fonte de informação desejada.Exemplo:
Google

TAG img

A tag <img> é usada para inserir imagens.
